# 3D Webcam Sigmoid Visualizer

A real-time computer vision application that transforms a live webcam feed into a dynamic 3D surface plot. The project processes pixel intensity through a Sigmoid activation function and quantizes the output into discrete steps for a "topographic" or "terraced" visual effect.
